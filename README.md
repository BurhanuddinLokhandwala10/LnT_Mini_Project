# LnT Mini Project

This repository contains the `LNT_ALL_PROJECTS` training workspace for the LnT DevOps program. It includes full end-to-end project implementations covering CI/CD, Docker containerization, Git workflow simulation, Infrastructure as Code with Terraform, and Kubernetes autoscaling.

## Contents

- `LNT_ALL_PROJECTS/` — main training workspace with all project folders
- `LNT_ALL_PROJECTS/README.md` — detailed project catalog and documentation

## About

This repository is organized to showcase hands-on DevOps solutions and learning milestones from the LnT Mini Project training program. Each project folder is self-contained and ready for review, testing, or deployment.
